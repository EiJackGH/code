## 2024-05-22 - CLI UX Delight
**Learning:** Even in backend scripts/CLI tools, visual hierarchy using ANSI colors and emojis significantly reduces cognitive load. Users can instantly scan for "Green" (Good/Buy) vs "Red" (Bad/Sell) without reading every line.
**Action:** For CLI tools that output logs or financial data, always implement a basic `Colors` class and use conditional formatting for key metrics (Status, Success/Failure, Profit/Loss). Explicitly align columns in summary reports for better readability.
